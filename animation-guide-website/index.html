前期制作核心环节

  • 剧本与分镜: 将文字故事转化为视觉蓝图，是整部影片的“说明书”。
  • 动态分镜: 针对复杂动作戏，用草稿动画清晰传达节奏与意图。
  • 角色与场景设计: 包含三视图、表情集、道具和场景氛围，统一作品的视觉风格。

这一阶段的核心是严谨的规划。一份详尽的前期设计是后续所有团队高效协作的根本保障。

中期制作核心环节

这是一个层层递进、高度依赖的瀑布式流程，每一步的质量都至关重要。

  1. Layout (构图): 分镜的细化版本，提供明确作画指示。
  2. 一原 (关键帧): 设计和绘制动作的核心帧，决定角色表演。
  3. 二原 (修型清线): 将一原草稿修正为干净、标准的最终线稿。
  4. 中割 (中间画): 补充关键帧之间的画面，使动作流畅。
  5. 上色: 根据设定为线稿填充颜色。

后期制作核心环节

  • 摄影 (合成): 将角色、背景、特效等所有图层合成，根据氛围图添加光影、景深等效果，营造最终画面感。
  • 声音制作: 包括音乐、音效和配音，赋予作品听觉生命力。
  • 剪辑: 通过对镜头节奏的把控，引导观众情绪，完成最终叙事。

后期是将所有独立素材整合升华的过程，最终诞生一部完整的“视听盛宴”。

团队角色

一个成功的项目离不开分工明确的团队。点击不同角色，了解他们在动画创作中的核心职责。

总制片

项目总负责人

负责项目管理、资源协调和对外沟通，需具备审美和卓越的沟通能力。

导演

创意总指挥

总揽全局，把控故事、节奏和艺术风格，确保作品的最终艺术完整性。

分镜师

视觉叙事者

将文字剧本转化为视觉画面，设计镜头语言和节奏。

美术

风格奠基人

确立并把控作品的整体视觉风格，包括色彩、场景、道具等。

一原画师

动作设计师

富有创造性的岗位，负责设计和绘制动作的关键帧，决定角色表演。

二原画师

造型执行者

侧重造型准确性和线条规范性，将一原草稿完善成干净的最终线稿。

作画监督

质量保证核心

修正原画，统一不同原画师的画风，确保角色在所有镜头中不“走形”。

中割/上色

新人入行起点

中割负责补充中间画使动作流畅；上色负责填色。是锻炼基本功的绝佳岗位。

职业发展路径

行业内主要存在两种发展模式：在单一领域深耕的“专精深化”，和跨职能学习的“跨界融合”。

两种核心发展路径

特点：风险低，成长轨迹清晰

在同一专业领域内纵向发展，目标是成为该领域的专家。需要长年累月的练习和积累。

二原画师 → 作画监督

一原画师 → 演出 → 导演

中割师 → 动检

特点：综合能力成长快

在不同岗位间横向流动，能学习多元技能，对流程有更全面的理解，为成为管理或导演打下基础。

中割 → 二原 (从动作辅助到造型绘制)

二原 → 一原 (从造型执行到动作设计)
